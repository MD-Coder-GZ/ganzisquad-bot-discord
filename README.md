Политика конфиденциальности GANZISQUAD BOT
Последнее обновление: 13.07.2026

1. Общая информация
GANZISQUAD BOT — это Discord‑бот, предназначенный для автоматизации управления сервером, предоставления статистики активности, логирования действий и улучшения взаимодействия пользователей.
Используя бота, вы соглашаетесь с данной Политикой конфиденциальности.

2. Какие данные собирает бот
Бот обрабатывает только те данные, которые необходимы для его работы:

Идентификаторы
Discord User ID

Server (Guild) ID

Channel ID

Message ID

Публичная информация профиля
Никнейм

Username

Аватар

Контент, переданный через команды
Сообщения, отправленные боту

Параметры команд

Медиа, отправленные через взаимодействие с ботом

Активность
Время в голосовых каналах

Количество сообщений

Типы отправленного контента (фото, ссылки, GIF, вложения и т.д.)

Бот не собирает и не обрабатывает чувствительные данные (пол, возраст, местоположение, личные сообщения, платежную информацию).

3. Использование Privileged Intents
Бот использует следующие Privileged Intents:

Guild Members Intent
Используется для:

корректного отображения профиля пользователя

статистики активности

логирования изменений ролей

работы системы уровней

Presence Intent
Используется для:

подсчёта времени в голосовых каналах

отслеживания активности участников

Message Content Intent
Используется для:

подсчёта сообщений

анализа типов контента

работы системы опыта

Бот использует эти интенты строго в рамках функционала, не собирает лишние данные и не анализирует личные сообщения.

4. Цели обработки данных
Данные используются для:

работы статистики активности

формирования таблиц лидеров

выдачи наградных ролей

логирования действий участников

корректной работы команд и меню

улучшения функционала бота

Данные не используются для рекламы, передачи третьим лицам или коммерческих целей.

5. Хранение данных
Бот может временно хранить данные в базе данных для обеспечения функционала.

данные могут быть удалены автоматически

данные могут быть очищены при обновлении бота

данные не хранятся дольше, чем это необходимо для работы функций

6. Передача данных третьим лицам
Бот не передаёт данные третьим лицам, за исключением:

Discord Inc. (как часть платформы)

хостинг‑провайдера (сервер, на котором работает бот)

Бот не использует сторонние рекламные сервисы и не продаёт данные.

7. Ответственность
Разработчик стремится обеспечить стабильную работу бота, однако:

бот предоставляется «как есть»

разработчик не гарантирует постоянную доступность сервиса

разработчик не несёт ответственности за действия Discord, хостинга или сторонних API

8. Удаление данных
Пользователь может запросить удаление данных, связанных с его Discord ID.

Удаление может быть невозможно, если:

данные необходимы для работы бота

невозможно подтвердить личность пользователя

данные уже удалены автоматически

9. Изменения политики
Политика может быть обновлена в любое время.
Продолжение использования бота означает согласие с новой версией.
