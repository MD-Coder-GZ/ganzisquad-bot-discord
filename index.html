Политика конфиденциальности GANZISQUAD BOT

Последнее обновление: 13.07.2026

1. Общая информация

GANZISQUAD BOT — Discord‑бот, предназначенный для автоматизации управления сервером, предоставления статистики активности, логирования действий и улучшения взаимодействия пользователей. Используя бота, вы соглашаетесь с данной Политикой конфиденциальности.

2. Какие данные собирает бот

Бот обрабатывает только данные, необходимые для его работы:

  • Discord User ID, Server ID, Channel ID, Message ID
  • Никнейм, username, аватар
  • Контент, переданный через команды
  • Активность: время в голосовых, количество сообщений, типы контента

Бот не собирает чувствительные данные.

3. Использование Privileged Intents

Guild Members Intent — статистика, уровни, логирование ролей.

Presence Intent — подсчёт времени в голосовых, активность.

Message Content Intent — подсчёт сообщений и типов контента.

Интенты используются строго в рамках функционала.

4. Цели обработки данных

Данные используются для статистики, лидербордов, наградных ролей, логирования действий и корректной работы команд.

5. Хранение данных

Данные могут временно храниться в базе данных и удаляться автоматически. Данные не хранятся дольше, чем необходимо.

6. Передача данных третьим лицам

Бот не передаёт данные третьим лицам, кроме Discord Inc. и хостинг‑провайдера.

7. Ответственность

Бот предоставляется «как есть». Разработчик не гарантирует постоянную доступность и не несёт ответственности за действия Discord или хостинга.

8. Удаление данных

Пользователь может запросить удаление данных, связанных с его Discord ID. Удаление может быть невозможно, если данные необходимы для работы бота или уже удалены автоматически.

9. Изменения политики

Политика может быть обновлена в любое время. Продолжение использования бота означает согласие с новой версией.
